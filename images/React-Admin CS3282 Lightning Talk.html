Problem Background

Problem Background

  • New doctors struggle to prioritise tasks in a ward
  • Traditional teaching methods struggle to teach this skill

Ward environments have

  • Lots of competing priorities and information
  • Time pressure
  • High stakes (patients' lives)

New doctors are placed into a new and difficult situation where mistakes can have really bad consequences!

How is this skill taught right now?

  • Classroom education : hard to model a dynamic situation with multiple patients
  • Simulation & role-playing : resource and logistically intensive
  • Learning on the job : risk to patients & pressure on senior residents

Approaching the problem through a serious medical game

  • More accurately model a dynamic situation in a ward
  • Extremely scalable; additional players can practice without additional resources
  • Safe environment to make mistakes

About Code Blue

Code Blue is a serious medical simulation game set in a hospital ward, where junior doctors learn to prioritise tasks to better serve their patients.

The first users of Code Blue are intended to be students studying medicine and are preparing for work in real hospital wards.

Code Blue's Key Features

  • Randomised patients with art that reflects a range of conditions, ages, demographics
  • Priority list of tasks that can be added to, rearranged, and executed in order
  • Patient state deteriorates or improves based on time and patient actions
  • Game is scored and players can understand how they did

Admin Dashboard for Code Blue

  • Doctors can update data at any time via the dashboard
  • Allows for quick testing of game balance and extensibility after the project is over
